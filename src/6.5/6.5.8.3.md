# Release notes Shopware 6.5.8.3

## Abstract

This patch release contains 20+ bug fixes.

## System requirements

* tested on PHP 8.1, 8.2 and 8.3
* tested on MySQL 8.0.33, MariaDB 10.4. 10.5, 10.11 & 11.0
